# Magnet Station

A Safari extension to send magnet links directly to your Synology Download Station.

## Features

- Click any magnet link to automatically add it to Download Station
- View active downloads with progress bars

## Setup

1. Open the project in Xcode
2. Build and run (⌘R)
3. Enable the extension in Safari → Settings → Extensions
4. Click the extension icon and enter your Synology IP and credentials

## Requirements

- macOS 10.15+
- Safari 14+
- Synology NAS with Download Station installed
